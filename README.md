# Facial Emotion Recognition System

Real-time detection and classification of human emotions from webcam using OpenCV and CNN.

## Emotions detected
Happy, Sad, Angry, Fear, Surprise, Disgust, Neutral

## Tech Stack
Python | OpenCV | CNN | Haar Cascade | NumPy

## How to Run
pip install opencv-python numpy
python emotion_detection.py

## How it works
1. Webcam feed captured in real time
2. Haar Cascade detects faces in each frame
3. CNN model classifies emotion from detected face
4. Predicted emotion displayed on screen live
